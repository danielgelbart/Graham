---------- Set up rails -----------
* Don't use 'sudo' with rvm

$ rvm gemset create rails238 # create a specific gem set
$ rvm ruby-1.9.2@rails238 # tell the gemset it is working with ruby 1.9.2
$ gem install rails -v 2.3.8
